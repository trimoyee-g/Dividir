Settlr is a full-stack web application built with Next.js, Tailwind CSS, Convex, and Clerk, designed to make group expense management seamless. It enables users to split expenses, track balances in real-time, and simplify repayments.


Features
- Real-time Group Expense Management : Users may add, edit, and split expenses across group members with instant balance updates.

- Debt Simplification Algorithm : Reduced unnecessary transactions with intelligent debt optimization.

- Authentication & User Management : Secured login and session handling powered by Clerk.

- AI-Powered Financial Insights : Users get personalized tips and summaries based on group spending habits.

- Automated Email Reminders : For pending debts.

- Visual Dashboards : Users understand who owes what, how much, and why – all at a glance.
